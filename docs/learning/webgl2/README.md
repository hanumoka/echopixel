# WebGL2 Learning Notes

EchoPixel 프로젝트에서 사용하는 WebGL2 기술 학습 노트입니다.

## 학습 목표

1. **2D Array Texture**: Multi-frame DICOM을 GPU에 효율적으로 저장
2. **Fragment Shader**: VOI LUT, 컬러맵 GPU 처리
3. **Single Canvas 아키텍처**: 다중 뷰포트 렌더링
4. **메모리 관리**: 텍스처/버퍼 생명주기
